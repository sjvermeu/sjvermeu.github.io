cvechecker 3.3 released

Date Mon 16 September 2013 By Sven Vermeulen Category Security Tags cvechecker / release

I just uploaded a new release of cvechecker to the project files. The release is a (long overdue) bugfix release, but includes two small enhancements: support standard input for the binary list (so you can pipe the output of one command to cvechecker) and the introduction of the CVECHECKER_CONFFILE variable to refer to another location for the configuration file.

Big thanks to Anne Mulhern for the various patches submitted!
